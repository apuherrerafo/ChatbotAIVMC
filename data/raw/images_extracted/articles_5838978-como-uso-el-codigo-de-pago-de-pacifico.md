# ¿Qué es el código de pago Pacífico?

El código de pago de Pacífico es un **valor alfanumérico de 6 caracteres** que se encuentra activo en el BCP y que **sirve para pagar el valor de la oferta** que acabas de ganar.

## Imagen 3

# ¿Cómo puedo pagar la oferta usando el código de pago?

Podrás realizar el pago usando los **siguientes canales**:

## 1. Banca Web o Móvil:

### a. Personas Naturales:

Podrás realizar el pago usando la Banca Web o Móvil siguiendo estos pasos:

• Dirigirse a **Pago de Servicios**.
• Buscar la opción: **Pacífico Seguros**.
• Elegir la opción **"23 Deducibles Autos Dólares"**.
• Ingresar **la placa del activo** como código de pago.
• Después del pago **adjuntar el comprobante** en el módulo Tu Actividad.

### b. Personas Jurídicas:

Podrás realizar el pago usando el TeleCrédito siguiendo los pasos:

• Dirigirse a la opción **Operaciones Monetarias**.
• **Pagos Varios**
• **Pago de Servicios**.
• En clasificación de pagos: **Seguros y Clínicas**.
• Empresa: **"Pacífico Seguros"**.
• Elegir la opción **"23 Deducibles Autos Dólares"**.
• Ingresar **la placa del activo** como código de pago.
• Después del pago **adjuntar el comprobante** en el módulo Tu Actividad.

## 2. En ventanilla:

• Indicar que desea realizar un **pago a la cuenta recaudadora de Pacífico Seguros**.
• Indicar la cuenta o servicio **"23 Deducibles Autos Dólares"**.
• Ingresar **la placa del activo** como código de pago.
• Después del pago **adjuntar el comprobante** en Tu Actividad.
